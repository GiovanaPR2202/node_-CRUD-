## Metodos http 

Get    ==> Buscar uma informação(Dado) 
Post   ==> Inserir e Criar uma informação(Dado)
Delete ==> Remover uma informação(Dado)
Put    ==> Alterar uma informação (Dado)
Patch  ==> Alterar uma informação(Dado)expecífica
